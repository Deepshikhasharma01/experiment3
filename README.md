
FSD-I Experiment 3.1
Title

Product Card Component using React Props

Description

This experiment focuses on creating a reusable ProductCard component in React. The component is designed to display product information dynamically using props. It includes modern CSS styling and conditional rendering to visually represent stock availability. The layout is responsive and follows component-based architecture principles.

Learning Outcomes

After completing this experiment, I learned:

How to create reusable components in React
How to use props for passing dynamic data
How to design responsive UI components
How to apply modern CSS styling techniques
How to implement conditional rendering in React
How to display stock status visually
How to structure React projects efficiently
