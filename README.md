Title

Product Card Component using React Props

Description

This experiment focuses on creating a reusable ProductCard component in React. The component is designed to display product information dynamically using props. It includes modern CSS styling and conditional rendering to visually represent stock availability. The layout is responsive and follows component-based architecture principles.

Learning Outcomes

After completing this experiment, I learned:

How to create reusable components in React
How to use props for passing dynamic data
How to design responsive UI components
How to apply modern CSS styling techniques
How to implement conditional rendering in React
How to display stock status visually
How to structure React projects efficiently

JavaScript
CSS

Class Hierarchy Implementation using Inheritance in React

Description

This experiment focuses on implementing a class hierarchy using Object-Oriented Programming concepts in JavaScript (ES6). A base class Person is created and extended by Student and Teacher subclasses. The program demonstrates inheritance, method overriding, and polymorphism. The class instances are rendered dynamically in a React UI to visualize the output.

Learning Outcomes

After completing this experiment, I learned:

How to create classes using ES6 syntax
How to implement inheritance in JavaScript
How to use method overriding in subclasses
How to demonstrate polymorphism
How to integrate OOP concepts with React
How to render dynamic data in UI components
How to structure code using class-based logic
